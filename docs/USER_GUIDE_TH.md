# คู่มือการใช้งานระบบ WarfarinPro (User Manual)

ยินดีต้อนรับสู่คู่มือการใช้งานระบบ **WarfarinPro** เครื่องมือสนับสนุนการตัดสินใจทางคลินิก (Clinical Decision Support) สำหรับการวางแผนขนาดยาวาร์ฟารินโดยแพทย์และเภสัชกร และแสดงผลตารางยาประจำวันแบบเข้าใจง่ายสำหรับผู้ป่วยและผู้ดูแล

---

## 1. บทนำและวัตถุประสงค์ (Introduction)

**WarfarinPro** พัฒนาขึ้นมาภายใต้แนวคิด **"เข็มทิศความปลอดภัยของผู้ป่วย (The Patient's Safety Compass)"** เพื่อแก้ไขปัญหาสองด้านหลักในการบำบัดรักษาด้วยยาวาร์ฟาริน:
1. **สำหรับบุคลากรทางการแพทย์**: ลดความผิดพลาดทางคณิตศาสตร์ในการคำนวณสัดส่วนขนาดยาประจำสัปดาห์ เพิ่มความเร็วในการทำงานในห้องตรวจที่มีคนไข้หนาแน่น
2. **สำหรับผู้ป่วยและผู้ดูแล**: แปลงรหัสยารวมสัปดาห์ที่เข้าใจยาก ให้กลายเป็นตารางแนะนำยาประจำวันพร้อมภาพเม็ดยาสีตรงตามแผ่นยาจริงที่ผู้ป่วยหยิบจับจำแนกได้ทันทีโดยไม่ต้องคาดเดา

> [!WARNING]
> **ข้อจำกัดความรับผิดชอบทางกฎหมาย (Legal Disclaimer)**
> ระบบ WarfarinPro เป็นเพียงเครื่องมืออำนวยความสะดวกในการสนับสนุนการตัดสินใจเท่านั้น ไม่ใช่อุปกรณ์สั่งยาอัตโนมัติหรือทดแทนการตรวจวินิจฉัยทางคลินิก การตัดสินใจและวิจารณญาณของผู้รักษาถือเป็นที่สิ้นสุด

## 2. การทำงานสำหรับบุคลากรทางการแพทย์ (Clinician Guide)

ส่วนนี้อธิบายขั้นตอนการประเมินผู้ป่วย การกรอกข้อมูล และการสร้างแผนการรักษารายสัปดาห์ในห้องตรวจ

### 2.1 การกรอกข้อมูลทางคลินิก (Clinical Inputs)
ให้ป้อนข้อมูลตามช่องนำเข้าหลัก ดังนี้:
* **Current INR (Alt+I)**: ค่าผลการตรวจอัตราการแข็งตัวของเลือดปัจจุบัน
* **Previous weekly dose (Alt+P)**: ปริมาณยาที่ผู้ป่วยทานต่อสัปดาห์เดิม (หน่วยมิลลิกรัม) เช่น ทานวันละ 5 มก. สัปดาห์ละ 7 วัน เท่ากับ 35 มก.
* **Target range (Alt+T)**: กำหนดเกณฑ์เป้าหมาย INR ตามข้อบ่งชี้การรักษา:
  * *Standard*: 2.0–3.0 (สำหรับข้อบ่งชี้ทั่วไป เช่น โรคหัวใจเต้นผิดจังหวะ AF, ลิ่มเลือดอุดตัน DVT)
  * *Mechanical valve*: 2.5–3.5 (สำหรับผู้ใส่ลิ้นหัวใจเทียมโลหะ)
  * *Custom range*: กำหนดเกณฑ์เป้าหมายเฉพาะบุคคลด้วยตนเอง
* **Clinic visit day (Alt+V)**: ระบุวันตรวจพบแพทย์ในสัปดาห์ปัจจุบัน (ใช้คำนวณวันงดยาสัปดาห์แรก)

### 2.2 ระดับการตรวจจับความปลอดภัยของระบบ (Safety Banners)
ระบบจะทำการเปรียบเทียบค่า INR และระดับความเสี่ยงเพื่อจัดประเภทแจ้งเตือนแพทย์โดยอัตโนมัติ:
1. **Normal (ปกติ - แถบสีเขียว)**: ค่า INR อยู่ในเป้าหมายอย่างปลอดภัย แนะนำรับประทานยาขนาดเดิมต่อ
2. **Caution (ข้อควรระวัง - แถบสีส้ม)**: ค่า INR อยู่นอกช่วงเป้าหมายเล็กน้อย แนะนำสัดส่วนการปรับขนาดยาเพิ่ม/ลดทีละน้อย
3. **Danger (อันตราย - แถบสีแดง)**: ค่า INR สูงหรือต่ำเกินเกณฑ์ปลอดภัยอย่างเห็นได้ชัด (เช่น INR ≥ 5.0) ระบบจะขึ้นคำแนะนำให้งดยาและแนะนำการใช้ยาแก้อาการแข็งตัว (วิตามิน K) ตามโปรโตคอลความปลอดภัย
4. **Hard Stop (ระงับชั่วคราว - แถบสีแดงเข้ม)**: กรณีพบภาวะวิกฤต เช่น **มีภาวะเลือดออกรุนแรง (Major bleeding)** หรือ **ตรวจพบการตั้งครรภ์ (Pregnancy)** ระบบจะตัดระบบการคำนวณออกและขึ้นเตือนให้ส่งตัวรักษาฉุกเฉินทันที

### 2.3 ตัวทำเครื่องหมายความปลอดภัย (Safety Flags)
แพทย์สามารถติ๊กเลือกประวัติการประเมินความเสี่ยงได้โดยตรงในหัวข้อ **Safety Flags** ซึ่งจะจัดแบ่งกลุ่มออกเป็นสองส่วนเพื่อลดความสับสน (Cognitive Load):
* **ภาวะเสี่ยงสูงและข้อห้ามชี้บ่ง (Critical Safety & Context)**: เลือดออกรุนแรง (Alt+B), ลิ้นหัวใจเทียม, ตั้งครรภ์, โรคตับ
* **ปฏิกิริยาระหว่างยา (Drug Interaction Flags)**: ยาแก้ปวด NSAIDs, ยาฆ่าเชื้อ/ยาปฏิชีวนะ, ยาคุมจังหวะหัวใจ (Amiodarone), ยากันชัก, สมุนไพร/อาหารเสริม, เครื่องดื่มแอลกอฮอล์

### 2.4 คีย์ลัดคีย์บอร์ดสำหรับการป้อนข้อมูลความเร็วสูง (Keyboard Shortcuts)
ออกแบบมาเพื่อคอมพิวเตอร์ตั้งโต๊ะในห้องตรวจที่ต้องการประหยัดเวลา แพทย์สามารถควบคุมระบบได้โดยแทบไม่ต้องจับเมาส์:

> [!NOTE]
> **สำหรับผู้ใช้ระบบ macOS (Mac)**: ท่านสามารถเลือกใช้ปุ่ม **Control (⌃)** หรือปุ่ม **Option (⌥)** ร่วมกับคีย์บอร์ดที่ระบุได้ เช่น กด `Ctrl+I` หรือ `Option+I` เพื่อลดการชนกันกับการพิมพ์ตัวอักษรพิเศษบน Mac

| คีย์ลัด | คำสั่งที่เกิดขึ้น |
|---|---|
| **Alt + I** | โฟกัสไปที่ช่องกรอกผลตรวจ INR ปัจจุบัน |
| **Alt + P** | โฟกัสไปที่ช่องกรอกขนาดยารวมสัปดาห์เดิม |
| **Alt + T** | เลือก/สลับเกณฑ์เป้าหมาย INR (Target Range) |
| **Alt + V** | เลือกวันมาตรวจคลินิก (Clinic Visit Day) |
| **Alt + B** | ติ๊กเปิด/ปิดประวัติมีเลือดออกรุนแรง (Major Bleeding) |
| **Alt + D** | เลือกเปอร์เซ็นต์ปรับขนาดยา (Selected Adjustment) |
| **Alt + F** | เลือกจำนวนวันงดยาในสัปดาห์แรก (First Week Hold Doses) |
| **Alt + S** | เปิดหน้าต่างสรุปสมุดยาสำหรับการคัดลอก (Booklet Summary Dialog) |
| **Alt + O** | เปิดสลับหน้าจอไปที่หน้าแสดงผลผู้ป่วย (Patient Viewer) |
| **Alt + C** | คัดลอกลิงก์คนไข้ (Patient Link) ส่งต่อได้ทันที (แสดงข้อความมุมขวาล่างลื่นไหล) |
| **Alt + H** | สั่งพิมพ์ฉลากยาหรือกระดาษแจ้งผลออกทางเครื่องพิมพ์ (Print) |
| **Enter** | สลับไปโฟกัสช่องข้อมูลถัดไป (INR -> Dose เดิม -> Target -> Visit -> ปรับยา -> งดยา -> ตารางแต่ละวัน) |
| **Shift + Enter** | ย้อนกลับไปโฟกัสช่องข้อมูลก่อนหน้า |
| **Escape** | กดปิดหน้าต่างสรุปสมุดยาลอย (Close Booklet Modal) |

### 2.5 การปรับแต่งและแก้ไขตารางยาแมนนวล (Dose Customization)
* ระบบจะคำนวณตัวเลือกขนาดยาและการงดยาเบื้องต้นให้โดยอัตโนมัติ 
* แพทย์สามารถคลิกเปลี่ยนขนาดยาในแต่ละวัน (จันทร์–อาทิตย์) ในหัวข้อ **Editable Maintenance Week** เพื่อความเหมาะสมเฉพาะบุคคล
* **ปุ่ม Reset to Suggestion**: หากแพทย์ทำการแก้ไขแมนนวลแล้วขนาดยารวมสัปดาห์คลาดเคลื่อน หรือต้องการย้อนกลับไปใช้ค่าคำนวณเดิมของระบบ สามารถกดปุ่ม **Reset to Suggestion (ไอคอนหมุนย้อนกลับ)** ใต้ตารางยาเพื่อคืนค่าได้ในคลิกเดียว

## 3. การทำความเข้าใจรหัสยา W-code

**W-code** คือรหัสย่อแบบกะทัดรัด (เช่น `W120-2H`) ที่ใช้อำนวยความสะดวกในการจดบันทึกลงแฟ้มประวัติหรือเขียนลงในสมุดประจำตัวผู้ป่วยแบบคุมยาวาร์ฟาริน:
* **รหัสสัปดาห์แรก (เช่น W120)**: แสดงปริมาณยาวาร์ฟารินรวมประจำสัปดาห์โดยคูณด้วย 10 เช่น `W120` หมายถึงปริมาณยารวมสัปดาห์ละ **12.0 มิลลิกรัม** (เกิดจากการรับประทานยาขนาด 3 มก. จำนวน 4 วัน)
* **รหัสงดยา (เช่น 2H)**: ย่อมาจาก **Hold** หมายถึงจำนวนวันงดยาที่คนไข้ต้องปฏิบัติทันทีในสัปดาห์แรกนับจากวันนัดตรวจคลินิก เช่น `2H` คือให้งดยาใน 2 วันแรกที่รับการประเมิน

## 4. การทำงานสำหรับผู้ป่วยและผู้ดูแล (Patient Viewer Guide)

ผู้ป่วยสามารถสแกน **QR Code** บนแผ่นกระดาษแจ้งเตือนยาหรือคลิกเปิดลิงก์ผ่านโทรศัพท์มือถือ เพื่อเข้าสู่โหมดหน้าจอของผู้ป่วย (**Patient Mode**) ซึ่งทำงานได้ดีแม้กำลังออฟไลน์ (ไม่มีอินเทอร์เน็ตที่บ้าน)

### 4.1 การจดจำสีเม็ดยาแบบ Double-Encoding
เพื่อป้องกันความสับสนของผู้สูงอายุหรือผู้ที่มีปัญหาตาบอดสี/สายตาเลือนราง ระบบจำลองภาพเม็ดยาที่มีรูปร่างและขนาดชัดเจนควบคู่ไปกับสีจริง:
* 🟠 **เม็ดยาสีส้มขนาดเล็ก**: หมายถึง ยาวาร์ฟารินขนาด **2 มิลลิกรัม** (มีตัวเลข 2 ตรงกลางชัดเจน)
* 🔵 **เม็ดยาสีฟ้าขนาดกลาง**: หมายถึง ยาวาร์ฟารินขนาด **3 มิลลิกรัม** (มีตัวเลข 3 ตรงกลางชัดเจน)
* 🔴 **แถบยาสีแดงคำว่า งดทานยา (HOLD)**: หมายถึง ให้งดรับประทานยาในวันดังกล่าว (ขนาดยา 0 มิลลิกรัม)
* 🌓 **สัญลักษณ์ครึ่งเม็ด (Half Pill)**: แสดงแถบเงาสีดำทึบโปร่งแสงบริเวณครึ่งขวาของตัวยา และขีดเส้นแบ่งซีกสีขาวโดดเด่นทำให้อ่านตัวหนังสือ contrast สีขาว `"1/2"` ได้ง่าย ไม่คลาดเคลื่อน

### 4.2 ตารางปฏิทินยาสองช่วงเวลา
* **สัปดาห์แรกหลังตรวจ (First Week)**: แสดงตารางการรับประทานยาโดยเริ่มนับตั้งแต่วันที่ตรวจแพทย์ในคลินิกเป็นวันแรก ซึ่งจะหักและงดทานยาตามแผนงดยาชั่วคราว (Hold Doses) เรียบร้อยแล้ว
* **ตารางระยะยาวปกติ (Maintenance Week)**: แสดงตารางปกติแบบวนซ้ำรายสัปดาห์ตั้งแต่วันจันทร์–อาทิตย์สำหรับทานในสัปดาห์ถัด ๆ ไป จนกว่าจะถึงกำหนดนัดครั้งใหม่

### 4.3 ระบบเสียงอ่านตารางยาไทย (Voice Guidance)
* ผู้สูงอายุหรือผู้ดูแลสามารถคลิกปุ่ม **"ฟังเสียงอ่านคำแนะนำภาษาไทย"** 
* ระบบสังเคราะห์เสียงจะอ่านสรุปวิธีกินยาวาร์ฟารินประจำวันให้ฟัง เช่น *"วันจันทร์ ทานยาวาร์ฟารินขนาด 2 มิลลิกรัม จำนวน 1 เม็ดสีส้มค่ะ"* เพื่อช่วยลดอัตราการอ่านฉลากยาผิดพลาด

### 4.4 การบันทึกและเปิดใช้งานแบบออฟไลน์
* ผู้ป่วยสามารถกดบันทึกหน้าตารางยานี้เก็บไว้ในเครื่องโทรศัพท์มือถือ (Save Plan)
* แผนยาจะถูกเก็บลงความจำของเว็บบราวเซอร์ทันที ทำให้คนไข้เปิดดูวิธีกินยาประจำวันได้ตลอดเวลาที่บ้านแม้ไม่เชื่อมต่ออินเทอร์เน็ต
